# Hacktoon-Aegis-Venus

Proyecto desarrollado para el reto OmegaHack 2026 enfocado en optimizar la gestión de PQRSDs para la Alcaldía de Medellín.

---

## Descripción

Esta solución busca mejorar el proceso de recepción, clasificación y análisis de solicitudes ciudadanas (PQRSDs) mediante el uso de inteligencia artificial, automatización y centralización de canales.

El sistema permite:
- Integrar múltiples canales (web, correo, redes, etc.)
- Pre-clasificar solicitudes automáticamente
- Generar resúmenes inteligentes
- Facilitar la gestión para funcionarios
- Mantener validación humana obligatoria
